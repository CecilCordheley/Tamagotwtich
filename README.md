<h1>TamagoTwitch </h1>
<h2>Défintion</h2>
<p>Il s'agit d'une classe javascript offrant les méthodes permettant 
l'implémentation et le déploiement d'un "animal virtuel" qui s'adapte en fonction des évenement du live</p>
<h2>configuration requise</h2>
<ul>
  <li>tmi.min.js</li>
  <li>ChatBot.js</li>
</ul>
<p>tamagotwitch ne peut fonctionner sans la bibliothèque <b>tmi.min.js</b>. Elle est nécessaire pour le déploiement du bot twitch servant à la gestion des commandes.</p>
<p>Elle fonctionne également avec ChatBot.js qui permet de déploiement et l'utilistation de <b>MyG4meBot</b></p>
